# Antigrav - Retro Casino FPS

## How to Play
Explore the dark, retro-styled casino. Interact with games, use items, and watch out for the guard.

### Controls
- **WASD**: Move
- **Mouse**: Look
- **Click**: Lock cursor / Interact with items
- **E**: Interact with world (Casino Games, Guard)
- **ESC**: Unlock cursor

### Items (Keys 1-4)
1.  **Cigarette**: Relax. Click to take a drag. Smoke trails as you walk.
2.  **Hand**: Slap! Click to slap. Can knock out the guard.
3.  **Heroin**: A trip. Click to use. Visual/Audio effects. **Fades to black and resets player after use.**
4.  **Knife**: Dangerous. Click to stab. Leaves blood stains and kills guard (with a scream).

### Features
- **Blackjack**: Play at the table. Win money.
- **Snake**: Play on the computer.
- **Guard**: He watches. You can take him out.
- **Atmosphere**: Dark, gritty, Quake-style graphics.

## Running the Game
Run the `play_antigrav.sh` script in this folder, or serve the directory with a web server and open `index.html`.
